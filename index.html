O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas da web. Ele permite que desenvolvedores organizem textos, imagens, links, tabelas, formulários e mídias na tela do navegador.

1. Trabalhando com Formulários em HTML

Os formulários permitem coletar dados dos usuários. São criados com a tag <form>, que pode conter campos como:

  • <input type="text"> - Texto
  • <input type="email"> - E-mail
  • <input type="password"> - Senha
  • <textarea> - Texto maior
  • <button> - Botões de envio

Esses elementos ajudam a criar áreas de login, pesquisas, cadastros e muito mais.

2. Estruturando seu HTML + Formatações

Uma página bem estruturada facilita a leitura e manutenção. Algumas tags importantes:

  • <h1> a <h6> - Títulos e subtítulos
  • <p> - Parágrafos
  • <strong> - Negrito
  • <u> - Sublinhado
  • <i> - Itálico
  • <mark> - Texto marcado

Além disso, usamos <br> para quebra de linha e <hr> para divisores.

3. Trabalhando com Mídias utilizando HTML

Com HTML, é possível inserir diversos tipos de mídia:

  • <img> - Imagens
  • <audio> - Áudio
  • <video> - Vídeos
  • <source> - Fontes de mídia

Exemplo:

<img src="imagem.jpg" alt="Descrição">
<video controls><source src="video.mp4" type="video/mp4"></video>

4. Criando Tabelas com HTML

As tabelas organizam dados em linhas e colunas. As principais tags são:

  • <table> - Tabela
  • <tr> - Linha
  • <td> - Coluna (dado)
  • <th> - Cabeçalho da coluna
  • <thead>, <tbody>, <tfoot> - Estrutura da tabela
  • <caption> - Título da tabela

Tabelas são úteis para exibir preços, horários, dados estatísticos e comparações.

5. Entendendo HTML Semântico

O HTML semântico utiliza tags com significados claros como:

  • <header>: Cabeçalho
  • <nav>: Navegação
  • <main>: Conteúdo principal
  • <section>: Seções do conteúdo
  • <article>: Artigo independente
  • <footer>: Rodapé

Isso torna a página mais acessível e fácil de entender para usuários e mecanismos de busca.
